الصورة القياسية للدالة التربيعية

الدوال التربيعية هي دوال غير خطية يمكن كتابتها على الصورة القياسية الموضحة أدناه. يسمى التمثيل البياني لها قطعاً مكافئاً (Parabola) ويشبه حرف U.

2ax + bx + c = (x)f

تطبيق ١: استخراج المعاملات

y = 3x² - 5x + 7

a = 3

يحدد اتجاه فتحة القطع المكافئ

خطوة ٢: تحديد b (معامل x)

b = -5

انتبه للإشارة السالبة!

خطوة ٣: تحديد c (الحد الثابت)

c = 7
